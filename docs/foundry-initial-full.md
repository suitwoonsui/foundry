Core Concept
Create a dedicated project funding page where community members can contribute to the development of individual products, games, tools, or marketing pushes. Instead of one single donation pool, each project has its own dedicated Sui wallet, making funding transparent and traceable.
This lets the community:

Watch progress happen in real time


Receive meaningful benefits tied directly to the project they funded

Project runway


Resource allocation


Development milestones per project
