# Path ranks reference (seven pillars)

Every path uses **three earned ranks** (low → high). Ranks are **per path only**—not a global leaderboard. Wording matches `apps/foundry/frontend/lib/contribution-paths.ts` (source of truth for the site).

### Pillar accents (site & docs alignment)

Path cards, individual path headers, and the home pillar strip use the same **`accentClass`** gradients from `contribution-paths.ts` (Tailwind `from-*` / `to-*` / `ring-*`). Semantic intent:

| Path | Accent idea |
|------|-------------|
| **The Heralds** | Rose → orange: heat, attention, social lift. |
| **The Council** | Violet → indigo: depth, judgment, strategy. |
| **The Orators** | Sky → cyan: airwaves, clarity, “on stage.” |
| **The Diplomats** | Emerald → teal: growth, trust, bridges between sides. |
| **The Forge** | Amber → brand ember: fire, metal, shipping. |
| **The Vault** | Zinc: neutral metal, treasury / runway without “gold rush” default. |
| **The Hearth** | Orange → deep amber: warmth, home, living in the product. |

---

## Consistency rules

- **Three ranks per path** — same depth of ladder everywhere; meaning differs by path.  
- **Earned, not purchased** — recognition policy is cultural/product; not pay-to-rank.  
- **No cross-path ordering** — Beacon does not “outrank” Architect; they are different lanes.
