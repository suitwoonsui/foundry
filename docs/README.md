## Vocabulary

- **Seven Pillars of the Foundry** — use for the umbrella / program framing only (includes **application users**, not only builders, funders, and amplifiers).
- **Path** — day-to-day language for a contribution or **product-participation** lane and progression (levels, onboarding, site `/paths`).
